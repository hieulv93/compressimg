# Analytics Reports

## Folder

```
analytics/
├── landing-pages/
│   └── download.csv   ← GA4: "Trang đích + chuỗi truy vấn"
└── keywords/
    └── download.csv   ← GA4: "Cụm từ tìm kiếm"
```

## Cách tải

**Landing pages:** GA4 → Reports → Engagement → Landing page + query string → Share → Download CSV → copy đè vào `landing-pages/`

**Keywords:** GA4 → Reports → Acquisition → Traffic acquisition → (chọn Organic Google Search) → Share → Download CSV → copy đè vào `keywords/`

Ghi đè file cũ, không cần đổi tên.
