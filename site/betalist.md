### Description (150–300 words — dùng cái này)
```
CompressImg is a free image compressor that runs 100% in your browser. No files are uploaded to any server — compression happens locally using a Web Worker and the Canvas API.

Upload a JPG, PNG, WebP, or HEIC photo, adjust the quality slider, and download the compressed result in seconds. Most images are reduced by 60–90% at quality 80 with no visible quality loss.

The tool also includes resize, crop, convert, and 60+ format-specific and platform-specific tools — all browser-based, all free.

Built for:
• Photographers who don't want to upload client photos to a random server
• Developers optimizing images for web before deploying
• Anyone who needs to compress a photo for a form, email, or social upload without installing an app

No sign-up. No watermarks. No daily limit. Works on desktop and mobile.
```
