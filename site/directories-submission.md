## LONG DESCRIPTION (dùng cho AlternativeTo, SaaSHub — cho phép nhiều text hơn)

```
CompressImg offers four free, browser-based image tools with no sign-up required:

**Compress Image** — Reduce JPG, PNG, and WebP file sizes by up to 90% using a smart compression algorithm with an adjustable quality slider (1–100). Ideal for web performance, email attachments, and social media.

**Resize Image** — Change image dimensions to any custom size or choose from built-in presets for Instagram, YouTube, Twitter, and more. Aspect ratio lock keeps proportions intact.

**Convert Image** — Convert between JPG, PNG, and WebP instantly. PNG and WebP output preserves transparency (alpha channel). JPG output automatically fills transparent areas with white.

**Crop Image** — Trim and cut images to any area with mouse or touch drag. Includes free-form cropping and fixed ratio presets (1:1, 16:9, 4:3, 3:2, 9:16) for social media and print.

All tools run 100% in your browser using the Canvas API and JavaScript Web Workers. No images are ever uploaded to any server — processing is instant and completely private. Works on desktop and mobile browsers.
```
